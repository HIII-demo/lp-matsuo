# LP: Google x 松尾・岩澤研究室

## プロジェクト概要
Google x 松尾・岩澤研究室の自治体向けAIソリューションのランディングページを作成する。

## 重要ルール
- 計画は `plan/` ディレクトリにまとめ済み（v2確定）
- 実装は `plan/02_ui-design.md` と `plan/01_page-structure.md` に基づく
- 技術: 単一HTML + Tailwind CSS CDN

## LPの要件
1. **取組概要**: Google x 松尾・岩澤研究室の紹介。金儲けではなく「日本の自治体・地方のため」という姿勢を前面に出す
2. **訴求ポイント（3つ程度）**: 営業資料ベース
3. **事例一覧**: 仮で作成（具体の文言は松尾研が関わる）
4. **導入〜検証の流れ**: 無料/有料の範囲、期間の説明
5. **問い合わせ**: CTAセクション
6. **トレーニング**: AIコネクトアカデミー、地域課題解決（サイバーセキュリティ、人材育成等）
7. **神奈川の動画**: 埋め込み候補

## 注意事項
- 画像・細かい要素は仮（プレースホルダー）で作成
- 叩き台として作成し、後から差し替え前提
- 綺麗でイケてるデザインを目指す
